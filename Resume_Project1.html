E-Store: Standalone GUI Application

Description:
A Java application that creates a standalone GUI application that simulates an e-store. We call our store Nile Dot Com. The application will allow the user to place in stock items into a shopping cart and once all items are included, total all costs (including tax), produce an invoice, and append a transaction log file.
